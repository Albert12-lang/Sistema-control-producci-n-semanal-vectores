
# Documentación del Sistema de Control de Producción

## 1. Nombre del Proyecto
Sistema de Control de Producción Semanal


## 2. Descripción General

El sistema permite simular y analizar la producción semanal de una empresa mediante el uso de vectores en JavaScript.

El encargado podrá:
1. Seleccionar la cantidad de días de producción.
2. Generar datos aleatorios de producción.
3. Mostrar la producción diaria.
4. Obtener estadísticas automáticas sobre la producción.


El sistema está desarrollado con:
HTML → estructura visual.
JavaScript → lógica y procesamiento de datos.

## 14. Conclusión

El programa podrá presentar aleatoriamente cantidades de los valores durante los siete días de la semana, después se genera una suma de todas las cantidades, ver cual día de la semana tiene una cantidad de mayor producción, menor producción, el promedio de las semana, los días sobre el promedio, los días críticos y si llega ha encontrar algún valor repetido.
